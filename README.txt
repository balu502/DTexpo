````DTExpo````

Утилита для работы с экспозициями детектирующих амплификаторов.


Установка
=========

Предполагается, что драйвер ДТхх предварительно был успешно установлен в системе.
В папке setup нужно инсталлировать gpg4win-vanilla-2.3.4.exe 
до первого запуска утилиты. Сама утилита запускается файлом Start.bat в папке DTExpo.
Для удобства быстрого старта можно вытащить ярлык из папки Temp\DTExpo.exe на 
рабочий стол.

Назначение
==========

Утилита предназначена для записи значений заводских экспозиций в прибор ДТхх.
Одновременно может быть подключено несколько приборов, при старте утилиты
сервис-инженером выбирается нужный из списка. Чтение из прибора установленных
ранее значений экспозиций производится по нажатию кнопки Read, вызов диалога 
ввода и записи новых значений - по нажатию Write.


Варианты использования
======================

На данный момент функционирует вариант работы с экспозициями в условиях 
сервисного обслуживания внутри офиса сервис-центра или с применением ноутбука
с выездом в лабораторию клиента. В дальнейшем будет возможен вариант удаленной 
установки утилиты на ПК клиента и удаленного подключения сервис-инженера, где 
предусмотрен парольный ввод и запись данных экспозиций.
